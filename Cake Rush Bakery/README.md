3. Project Goals and Objectives

Online Orders:
Create a website with an e-store functionality that allows users to order cakes online with ease.

Brand Visibility:
Develop a visually appealing and user-friendly website to showcase the bakery’s brand, story, and background.

Customer Engagement:
Include Google Maps integration, a Google Form, and social media links to improve user interaction and trust.

Marketing and Growth:
Add promotional sections to attract new customers and retain existing ones through updates and discounts.


4. Justification of Design Choices

The overall design was created to highlight the bakery’s warm and creative personality. The colour palette, typography, and layout were chosen carefully to align with brand identity and improve user experience.

Colour Choices:

* Pink (#E21C47): Represents sweetness, joy, and friendliness — ideal for desserts.
* White (#FFFFFF): Suggests cleanliness and freshness, essential for food-related businesses.
* Soft Pink Background (#FCDDEA): Creates an inviting and calming atmosphere.

Typography Choices:

* Headers: Poppins Semi-Bold for a modern and readable design.
* Body Text:Open Sans for clean and professional readability.
* Decorative Text (Slogan): Pacifico or Dancing Script for personality and flair.

Layout and Navigation:
The website includes a clear and consistent structure:

* The header includes a logo, slogan, and navigation bar.
* The main section contains featured content and imagery.
* The footer provides quick links, contact details, and social media icons.

5. Website Features and Functionality

Homepage:
Includes a hero image, slogan, featured products, and a quick-order button that links to the products page.

Menu/Products Page:
Displays cakes, cupcakes, and pastries with prices, images, and descriptions. Includes an “add to cart” placeholder for future online orders.

Gallery Page:
Showcases high-quality images of custom cakes. Clicking on an image enlarges it using JavaScript functionality.

About Us Page:
Highlights the bakery’s history, vision, and mission. Includes customer testimonials, social media links, and an embedded Google Map.

Contact Page:
Contains a functional contact form, Google Maps integration, and social media links for easy communication.

Enquiry Page:
Allows customers to make custom cake or catering requests, validated using JavaScript before submission.

7. Technical Implementation
Languages Used: HTML5, CSS3, JavaScript

JavaScript Features:
* Image gallery with lightbox effect
* Form validation for both the contact and enquiry pages
* Navigation hover colour animation
* Dynamic content updates and smooth scrolling
* Lazy image loading for improved page speed

SEO Enhancements:

* Added `robots.txt` to instruct search engine crawlers which pages to index
* Created `sitemap.xml` to guide search engines through the website structure
* Added meta tags and alt text for accessibility
* Optimized images and deferred JavaScript for faster page loading

Security and Optimization:

* Basic input validation for all forms
* HTTPS compatibility
* Compressed media for performance
